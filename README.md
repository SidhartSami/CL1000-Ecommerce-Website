## About The Project

[![Product Name Screen Shot][product-screenshot]](https://sidhartsami.github.io/Ecommerce-Website_IICT)

Ozone is a static e-commerce website developed as part of the first-semester IICT project at FAST NUCES. Inspired by the sleek design philosophy of Zara, this project showcases a premium men's fashion brand with a minimal, modern interface.

Here's why this project stands out:

* Demonstrates core front-end development skills with HTML5, CSS3, and JavaScript
* Provides a complete e-commerce user experience flow from browsing to checkout
* Implements responsive design principles with organized, maintainable code structure
* Serves as an excellent portfolio piece for web development positions
* Showcases understanding of user interface design and user experience best practices

This project was created to master fundamental web development concepts and build a strong foundation in front-end technologies.

## Features

* **Fully Static Front-End Design** - Pure HTML, CSS, and JavaScript implementation
* **User Authentication Pages** - Login and Registration interfaces
* **Hero Landing Page** - Brand introduction with engaging visuals
* **Product Listing Page** - Browse and preview fashion items
* **Product Detail Pages** - In-depth product information with images and descriptions
* **Shopping Cart System** - Add/remove items from cart
* **Checkout Flow** - Complete purchase workflow
* **Responsive Styling** - Fully styled using internal CSS and image assets
* **Organized File Structure** - Clean, maintainable project architecture

## Getting Started

Follow these simple steps to get a local copy of the project up and running.

### Prerequisites

You only need a modern web browser to run this project. No additional software or dependencies are required.

* A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/SidhartSami/CL1000-Ecommerce-Website.git
   ```
2. Navigate to the project directory
   ```sh
   cd CL1000-Ecommerce-Website
   ```
3. Open the `HomePage.html` file in your web browser
   ```sh
   # On macOS
   open HomePage.html
   
   # On Windows
   start HomePage.html
   
   # Or simply double-click the file in your file explorer
   ```

Alternatively, you can view the live demo at: [https://sidhartsami.github.io/Ecommerce-Website_IICT](https://sidhartsami.github.io/Ecommerce-Website_IICT)

<!-- USAGE EXAMPLES -->
## Usage

**Browsing Products:**
1. Start from the landing page (HomePage.html)
2. Navigate to the product catalog using the navigation menu
3. Click on any product to view detailed information

**Making a Purchase:**
1. Add items to your cart from the product detail page
2. Review your cart on the Cart page
3. Proceed to checkout to complete your purchase

**User Account:**
1. Create a new account using the SignUp page
2. Login with your credentials on the Login page
3. Access your account information

For more detailed information, visit the [live demo](https://sidhartsami.github.io/Ecommerce-Website_IICT).
